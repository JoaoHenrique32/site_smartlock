Guia de Implantação

Este é o manual de execução oficial do SmartLock. Siga a ordem de inicialização abaixo para replicar e conectar todas as camadas do ecossistema distribuído na sua máquina local.

Passo a Passo de Execução

1. O Coração da Rede: Broker MQTT (Docker)

O servidor EMQX, equipado com os certificados de criptografia mTLS, deve ser o primeiro sistema a subir para rotear as mensagens.

cd docker/emqx
docker compose up -d

Painel de monitoramento do broker acessível em localhost:18083.

2. Motor de IA e Backend (Python/Django)

Responsável por rodar o DeepFace e processar os frames da câmera web em tempo real.

  • Crie o ambiente virtual e instale as bibliotecas de Visão Computacional:
    • pip install -r requirements.txt
  • Navegue até a pasta raiz do site e inicie o servidor:
    • python manage.py runserver

3. Painel de Administração (App Flutter)

Levante o aplicativo nativo para gerenciar aprovações pendentes e visualizar a galeria do banco de rostos.

  • Em um novo terminal, navegue até a pasta do aplicativo e execute:
    • flutter run -d windows

4. Dispositivo Físico (NodeMCU / ESP8266)

Compile e grave o firmware em C++ na placa responsável pelo acionamento da porta.

  • Abra a pasta do hardware no VS Code com a extensão PlatformIO.
  • No arquivo secrets.h, atualize a variável do IP do servidor MQTT (o endereço IPv4 da sua máquina atual).
  • Conecte o NodeMCU via cabo USB e clique no botão de Upload na barra inferior.
